Gitolite 服务架设

使用 git 下载 Gitolite 的源代码 git clone git://github.com/ossxp-com/gitolite.git

tolite/src 目录，执行安装 

cd gitolite/src 
./gl-easy-install git server admin

命令 gl-easy-install 的第一个参数 git 是服务器上创建的专用帐号ID，第二个参数 server 是服务器IP或者域名，第三个参数 admin 是管理员ID

编辑 .gitolite.rc 文件

管理 Gitolite

当 gitolite 安装完成后，在服务器端自动创建了一个用于 gitolite 自身管理的 git 库: gitolite-admin.git当 gitolite 安装完成后，在服务器端自动创建了一个用于 gitolite 自身管理的 git 库: gitolite-admin.git
