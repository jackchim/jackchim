mysql 安装

yum -y install mysql-server

将mysql放入到启动服务列表中 

chkconfig --add mysqld 
chkconfig mysqld on

编译参数 

./configure --prefix=/usr/local/mysql --with-extra-charsets=complex --enable-thread-safe-client --enable-assembler --with-mysqld-ldflags=-all-static --with-charset=utf8 --enable-thread-safe-client --with-big-tables --with-readline --with-ssl --with-embedded-server --enable-local-infile --with-plugins=innobase

基本命令

库的操作

  • 显示数据库 SHOW DATABASES;
  • 使用数据库 USE <DATABASENAME>;
  • 删库 drop database 库名;
  • 删除名为test的数据库 DROP DATABASE TEST;
  • 创建数据库 CREATE DATABASE <NAME>;

表的操作

  • 查看数据库中已存在的表 SHOW TABLES;
  • 显示数据表的结构 describe 表名;
  • 显示表中的记录 select * from 表名;
  • 建表 use 库名； create table 表名;
  • 删表 drop table 表名；
  • 将表中记录清空 delete from 表名;
  • 重命名表 alter table t1 rename t2;

字段操作

  • 增加一个字段 alter table table_name add column column_name type default value;
  • 更改一个字段名字 alter table table_name change sorce_col_name dest_col_name type default value;
  • 改变一个字段的默认值 alter table book alter flag set default '0';
  • 改变一个字段的数据类型 alter table userinfo change column username username varchar(20);
  • 删除字段 alter table form1 drop column 列名;

mysql 信息

  • 查询时间 select now();
  • 查询当前用户 select user();
  • 查询数据库版本 select version();
  • 查询当前使用的数据库 select database();

设置root密码 set password for root@localhost=password('root');

导出数据 mysql -uroot -p dbname > db.sql
