高亮光标所在行（我选择不改变前景和背景色，只加了一条下划线）： 

set cursorline
hi cursorline guibg=NONE gui=underline

注：高亮当前行在一定程度上会影响 Vim 的性能，降低缓冲区更新速度

让 Vim 的默认寄存器和系统剪贴板共享，可以在 vimrc 中加入： 

set clipboard+=unnamed

Vim 启动时自动进入插入模式，可以在 vimrc 中加入： 

exe "startinsert"

映射(Mapping)

" Normal Mode, Visual Mode, and Select Mode,
" use <Tab> and <Shift-Tab> to indent
" @see http://c9s.blogspot.com/2007/10/vim-tips.html
nmap <tab> v>
nmap <s-tab> v<
vmap <tab> >gv
vmap <s-tab> <gv

注意：normal 模式下 <Tab> 键默认用于跳转到跳转列表较新的位置，我已经取消了这个设置，毕竟 normal 模式下基本是使用 >>。

默认隐藏菜单栏和工具栏，可以通过 <F12> 切换显示和隐藏。 

" Toggle Menu and Toolbar
" @see http://liyanrui.is-programmer.com/articles/1791/gvim-menu-and-toolbar-toggle.html
set guioptions-=m
set guioptions-=T
map <silent> <F2> :if &guioptions =~# 'T' <Bar>
\set guioptions-=T <Bar>
\set guioptions-=m <bar>
\else <Bar>
\set guioptions+=T <Bar>
\set guioptions+=m <Bar>
\endif<CR>

删除vim中的^M字符

使用替换命令 :%s/^M//g 注意这个^M是分别按键盘上的ctrl+v和ctrl+m来生成的。

录制动作宏

qa开始录制宏，编辑器下方提示：recording

q完成录制

3@a进行宏回放，重复3次录制名为a 的动作.

en

  • 1. Create the first list entry, make sure it starts with a number.
  • 2. qa - start recording into register 'a'
  • 3. Y - yank the entry
  • 4. p - put a copy of the entry below the first one
  • 5. CTRL-A - increment the number
  • 6. q - stop recording
  • 7. <count>@a - repeat the yank, put and increment <count> times

编译支持python ruby

cd vim73/src
./configure --enable-multibyte --enable-xim --without-x --enable-gui=no --enable-pythoninterp --enable-python3interp  --enable-rubyinterp --enable-cscope  --with-features=huge
make && make install

no terminal library found

You need to install a terminal library; for example ncurses.
Or specify the name of the library with –with-tlib.

提示没有安装terminal library，cygwin 下安装ncurses libncurses. OK!
